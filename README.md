# ⚡ Getting Started

## Clone Repository

```bash
git clone https://github.com/devlife007/IYG-Portfolio.git
```

## Navigate Into Project

```bash
cd IYG-Portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

---

# 🏗️ Build For Production

```bash
npm run build
```

---

# 🌐 Deployment

This project is deployed using Netlify.

## Netlify Configuration

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```

Example `netlify.toml`:

```toml
[build]
  command = "npm run build"
  publish = "dist"

[build.environment]
  NODE_VERSION = "22"
```
